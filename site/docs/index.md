## 站点定位

这不是单纯的“课件目录”，也不是单纯的“交互 demo 集合”。

这里更像一本逐步成长的课程教材站点：

- 教材正文负责叙事主线
- 静态图与代码块负责稳定表达
- 交互演示负责少量关键概念的动态理解

## 如何使用这个站点

- 如果你想按章节阅读，请进入“教材”
- 如果你想直接体验交互页面，请进入“交互演示”
- 如果你是在 GitHub 仓库里看到某个 `index.html` 文件，请不要直接在文件页里阅读源码，而是通过本站入口打开对应页面

## 当前已接入的内容

- 一章教材正文草稿：算法复杂度
- 多个交互式演示：二分查找递归、归并排序、循环队列、括号匹配、斐波那契递归

## 后续建议

优先把复杂度、线性结构、栈与队列整理成前三章，再逐步把现有交互页面嵌入到对应章节中。
